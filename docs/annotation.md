## Annotation

This project focuses on the development of a decentralized network and blockchain system for processing financial transactions. The goal is to create a reliable and secure system that enables protected data exchange between nodes while ensuring transaction integrity.

The project implements security mechanisms based on cryptographic methods, as well as consensus algorithms that ensure data consistency across network nodes. The system architecture includes a desktop application for node management and a mobile Android application designed for user interaction with the network.

The developed system is aimed at improving data protection, increasing resistance to attacks, and decentralizing transaction processing, making it relevant for use in the financial sector and other security-critical domains.
